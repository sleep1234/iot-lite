# IoT Lite - 物联网卡管理工具（精简版）

> 中国电信物联网 CRM 系统的精简管理工具，核心理念：**智能输入 + API 自动串联 = 最少操作步骤**

---

## ✨ 特性

- **智能查询** — 粘贴 ICCID 或接入号，自动识别并一步查完所有信息
- **批量操作** — 多个号码一行一个，批量查询/解绑
- **完整详情** — 客户信息、卡片状态、IP/DNN/IMEI、已订购服务、扩展属性、定向编码
- **快捷操作** — 查询结果页直接执行解绑/复机
- **工单查询** — 按天数+员工筛选历史工单
- **Session 自动管理** — TGT 刷新 + 全自动登录

## 🚀 快速开始

### 环境要求

- Python 3.10+
- 内网环境（能访问 CRM 系统）

### 安装

```bash
# 从原项目复制必要文件
cp /path/to/merged_app/xsign.py .
cp /path/to/merged_app/config.py .
cp /path/to/merged_app/auto_login.py .
cp /path/to/merged_app/des_crypto.py .
cp -r /path/to/merged_app/static/ .

# 安装依赖
pip install -r requirements.txt
```

### 配置

编辑 `config.py` 或设置环境变量：

```env
IOT_SECRET_KEY=your-secret-key
AI_CAPTCHA_API=http://your-ai-api/v1
AI_CAPTCHA_KEY=your-api-key
CRM_PHONE=1xxxxxxxxxx
```

### 启动

```bash
python app.py
```

访问 `http://localhost:10001`

## 📖 使用说明

### 智能查询

在搜索框中粘贴以下任一格式：

| 输入格式 | 自动识别 | 处理流程 |
|----------|----------|----------|
| `8986...` 19-20位 | ICCID | → 查接入号 → 查客户 → 查产品 → 查详情 |
| `141...` 13位 | 接入号 | → 直接查客户 → 查产品 → 查详情 |
| 多行（换行分隔） | 批量 | → 自动切换到批量模式 |

### 查询结果包含

- **客户信息** — 名称、编号、地址
- **卡片信息** — 状态、套餐、IP、DNN、IMEI、UIM号、绑定状态
- **产品实例** — 所有产品及状态
- **已订购服务** — 所有子服务及属性（切片、SA模式、VPDN等）
- **扩展属性** — 集团编号、联系号码、白名单类型、流量上限
- **定向编码** — 4G/5G定向编码、APN

### 快捷操作

查询完成后可直接执行：

- 🔓 **机卡解绑** — 单卡解绑（完整9步流程）
- 🔄 **违章复机** — 违章停机卡恢复
- 📋 **复制** — 接入号/ICCID/全部JSON

### 批量操作

- **批量查询** — 多个号码一行一个
- **批量解绑** — 粘贴多个ICCID/接入号，一键解绑

### 工单查询

- 按天数筛选（今天/3天/7天/30天）
- 按员工姓名筛选
- 显示工单号、业务、客户、状态、时间、经办人

## 🔐 认证说明

### 老系统 (ctc-ctiot:8043)

- **签名**: `X-Sign = MD5(body + timestamp)`
- **Cookie**: `PSESSIONNEWID=<session_id>`
- **Session 来源**: CRM SSO → TGT → PSESSIONNEWID

### Session 生命周期

```
PSESSIONNEWID 过期
    → TGT 刷新（自动）
        → 成功：无感续期
        → 失败：全自动登录
            → 图形验证码 OCR → 短信 → 登录
            → 成功：无感续期
            → 失败：提示手动连接
```

## ⚠️ 注意事项

1. **内网环境** — 必须在能访问 CRM 系统的内网环境运行
2. **Session 有效期** — PSESSIONNEWID 通常 8 小时有效
3. **操作风险** — 解绑/复机操作不可撤销，请确认后再执行
4. **并发安全** — Flask 多线程环境下 Session 管理已做线程安全处理
